Aprendendo a criar listas

Listas que não seguem uma ordem: Utiliza-se "ul"

  • Item 1
  • Item 2
  • Item 3

Listas que seguem uma ordem: Utiliza-se "ol"

  1. Item 1
  2. Item 2
  3. Item 3
E para criar intens na lista utiliza-se "li"

Aprendendo a criar links

É Necessário utilizar a letra "a" para ancorar algo, que ira me direcionar para algum lugar;
Dentro dessa tag 'a', para criar um link usa-se "href" que ao clicar, abrirá o link escolhido na própria aba;
Já para o lik ter acesso em uma nova aba, usa-se " target="_blank" ".

Aprendo a criar imagens

A tag da imagem é "img", entretano, ela não tem uma tag de fechamento
e para ser utilizada, após criar a tag de abertura, temos que criar o comando "src="

Aprendendo sobre quebra de linhas

Basicamente, para quebrar a linha dentro de um parágrafo "p"
Utiliza-se "br"

Já para crirar uma linha, como no exemplo dessa mesma página la em cima
Uma linha horizontal que facilita a divisão de um parágrafo para o outro, utiliza-se "hr".

Por fim, para uma busca mais abrangente de tags e saber qual utilizar para uma determinada
necessidade, acesse o site de exemplo developer.mozilla
